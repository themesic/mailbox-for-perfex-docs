## ğŸ“§ How Threading Works

### Thread Structure

When threading is enabled:
- Related emails are grouped together
- Original email appears first
- Replies appear below in chronological order
- Thread can be expanded or collapsed

### Thread Indicators

- **Thread Icon**: Shows number of messages in thread
- **Expand/Collapse**: Click to show/hide thread messages
- **Unread Count**: Shows unread messages in thread

---

## ğŸ” Viewing Threads

### Expand Thread

1. Find an email with threading indicator
2. Click the **expand icon** (usually a chevron or arrow)
3. All related messages in the thread will appear
4. Messages are shown in chronological order

### Collapse Thread

1. Click the **collapse icon** again
2. Thread collapses to show only the latest message
3. Thread indicator shows total message count

### Thread Navigation

- **Previous/Next**: Navigate between messages in thread
- **Original Message**: Jump to the first email in thread
- **Latest Reply**: Jump to the most recent message

## âœ‰ï¸ Replying in Threads

### Reply to Thread

1. Open the thread
2. Click **Reply** on any message in the thread
3. Your reply will be added to the thread
4. Thread automatically expands to show your reply

### Reply All in Thread

1. Open the thread
2. Click **Reply All**
3. All participants in the thread will receive your reply
4. Reply is added to the thread

## ğŸ”„ Thread Behavior

### New Emails in Thread

- New replies automatically appear in the thread
- Thread moves to top of inbox (if sorted by date)
- Unread count updates automatically

### Thread Sorting

- Threads are sorted by the latest message date
- Most recent activity appears first
- Original message date is preserved

## ğŸ’¡ Best Practices

1. **Keep Threads Together**: Don't delete individual messages from threads
2. **Use Thread View**: Expand threads to see full conversation context
3. **Reply in Context**: Reply within threads to maintain conversation flow
4. **Tag Threads**: Tag entire threads for organization
5. **Archive Threads**: Archive complete threads when conversation is done
