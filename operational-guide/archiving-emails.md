Archiving is different from deleting:
- **Archive**: Moves email out of inbox but keeps it accessible
- **Delete**: Permanently removes email (moved to Trash)

Archived emails can be:
- Searched
- Restored to inbox
- Accessed from the Archive folder
- Tagged and organized

1. Select multiple emails using checkboxes
2. Click the **Actions** dropdown menu
3. Select **Archive**
4. All selected emails will be archived

### Method 3: Keyboard Shortcut

1. Select the email(s)
2. Press **A** key (if keyboard shortcuts are enabled)
3. Email(s) will be archived
