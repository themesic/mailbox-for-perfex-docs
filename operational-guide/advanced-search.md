## ğŸ“‹ Search Criteria

### From (Sender)

Search by email sender:
- Enter email address: `john@example.com`
- Enter domain: `@example.com`
- Enter name: `John Smith`

**Examples:**
- `support@` - Find all emails from support addresses
- `@company.com` - Find all emails from company domain

---

### Subject

Search in email subject lines:
- Enter keywords: `invoice`, `urgent`, `meeting`
- Use partial matches: `inv` finds "invoice", "invitation", etc.

**Examples:**
- `invoice` - Find all emails with "invoice" in subject
- `[URGENT]` - Find urgent emails

---

### Body (Content)

Search in email body/content:
- Enter keywords or phrases
- Searches through entire email text

**Examples:**
- `project deadline` - Find emails mentioning project deadline
- `payment received` - Find payment confirmations

---

### To (Recipient)

Search by recipient:
- Your email address
- Other recipients in To/CC fields

---

### Tags

Filter by email tags/labels:
- Select one or more tags from dropdown
- Shows only emails with selected tags

**Examples:**
- Select "Support" tag - Shows only support emails
- Select multiple tags - Shows emails with any selected tag

---

### Date Range

Filter by date:
- **From Date**: Start of date range
- **To Date**: End of date range
- Select dates using date picker

**Examples:**
- Last week: Set From/To dates for last 7 days
- This month: Set From/To dates for current month
- Specific period: Set custom date range

---

### Read Status

Filter by read status:
- **Read**: Show only read emails
- **Unread**: Show only unread emails
- **All**: Show both read and unread

---

### Has Attachment

Filter by attachment presence:
- **Yes**: Show only emails with attachments
- **No**: Show only emails without attachments
- **All**: Show both

## ğŸ’¡ Search Tips

### Combine Criteria

Use multiple criteria for precise results:
- **From**: `support@` AND **Subject**: `urgent` = Support urgent emails
- **Tags**: "VIP" AND **Date**: Last 7 days = Recent VIP emails
- **Has Attachment**: Yes AND **Read Status**: Unread = Unread emails with attachments

### Use Partial Matches

- `inv` matches "invoice", "invitation", "inventory"
- `@company` matches all emails from company domain
- `support` matches "support", "Support", "SUPPORT" (case-insensitive)

### Date Ranges

- Use specific dates for precise filtering
- Use date ranges to find emails from periods
- Combine with other criteria for better results
